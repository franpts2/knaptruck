Reads pallet data from a CSV file.

Parameters
filenamePath to the CSV file containing pallet data
palletsArray to store pallet IDs
weightsArray to store pallet weights
profitsArray to store pallet profit values
Note
The CSV file should have a header row and columns for pallet ID, weight, and profit, separated by commas. The function skips the header row.
