unsigned int knapsackDP ( unsigned int profits[],
unsigned int weights[],
unsigned int n,
unsigned int capacity,
bool usedItems[] )

Dynamic programming solution for the 0/1 Knapsack problem.

Parameters
profitsArray of profit values for each pallet
weightsArray of weight values for each pallet
nNumber of pallets
capacityMaximum weight capacity of the truck
usedItemsOutput array that will indicate which pallets were selected
Returns
The maximum total profit achievable
Note
When multiple solutions have the same profit, solutions with fewer pallets are preferred. If pallet counts are equal, solutions with pallets having lower indices are prioritized.
Time Complexity: O(n×W) where n is the number of pallets and W is the truck capacity
Space Complexity: O(n×W) for the DP table
