Runs both greedy approximations and returns the better solution.

Parameters
profitsArray of profit values for each pallet
weightsArray of weight values for each pallet
nNumber of pallets
max_weightMaximum weight capacity of truck
Returns
GreedySol containing the better solution
Note
When comparing solutions, the function prioritizes higher total profit. If profits are equal, it selects the solution with fewer pallets. If pallet counts are also equal, it chooses the solution with lower total weight.
Time Complexity: O(n log n) for each greedy approach = O(n log n)
Space Complexity: O(n) for storing solutions
